如何用 Hexo 搭建个人博客
博客搭建 2024/12/22

写博客是一种记录生活、分享知识的方式，同时也是展示个人能力的舞台。今天，我将以 Hexo 为例，带你完成从安装到部署的完整过程，帮助你搭建一个属于自己的博客。

1. 安装 Hexo

Hexo 是基于 Node.js 的静态博客框架，所以在开始之前，你需要完成以下准备：

1.1. 安装 Node.js

打开 Node.js 官网。
下载并安装最新版 Node.js（建议使用 LTS 版本）。
安装完成后，在终端中运行以下命令检查版本：

node -v
npm -v

如果输出类似以下内容，说明安装成功：

v16.20.1
8.19.2

1.2 安装 Git

打开 Git 官网。
下载并安装 Git。
安装完成后，在终端中运行以下命令检查版本：

git --version

输出类似以下内容，说明安装成功：

git version 2.34.1

1.3 安装 Hexo

在终端中运行以下命令全局安装 Hexo CLI：

npm install -g hexo-cli

安装完成后，运行以下命令确认安装成功：

hexo -v

输出类似以下内容，说明安装成功：

hexo: 6.3.0
hexo-cli: 4.2.0

2. 初始化博客

2.1 在终端中进入你希望存放博客的目录，例如：

cd ~/Documents

2.2 运行以下命令初始化一个新的 Hexo 项目：

hexo init my-blog
cd my-blog
npm install

2.3 结构说明：

scaffolds：用于创建文章的模板文件。
source：存放博客的主要内容（如文章和图片）。
themes：主题文件夹，存放博客的主题文件。
_config.yml：Hexo 的全局配置文件。

3. 本地预览博客

3.1 启动 Hexo 本地服务器

在终端中运行以下命令，启动本地服务器：

hexo server

3.2 访问本地博客

打开浏览器，访问以下地址：

http://localhost:4000

如果看到 Hexo 的默认页面，说明本地博客运行成功。

4. 创建文章

4.1 使用以下命令创建新文章：

hexo new post "你的文章标题"

4.2 编辑文章内容

文章文件会被保存在 source/_posts 文件夹中，使用编辑器打开 Markdown 文件，并按照以下格式编辑：

---
title: 你的文章标题
date: 2024-12-22
tags:
  - 示例标签
  - 教程
---
这里是你的文章内容。

5. 更换主题

5.1 查找主题

在 Hexo 主题列表 中浏览并选择一个你喜欢的主题。

5.2 下载主题

进入博客目录，在 themes 文件夹中下载并解压主题，例如：

git clone https://github.com/username/theme-name themes/theme-name

5.3 配置主题

打开 Hexo 的全局配置文件 _config.yml，找到 theme，并修改为你下载的主题名称，例如：

theme: theme-name

6. 部署到 GitHub Pages

6.1 创建 GitHub 仓库

登录 GitHub，创建一个新的仓库，命名为 yourname.github.io。

6.2 安装部署工具

在 Hexo 项目目录下，安装部署插件：

npm install hexo-deployer-git --save

6.3 配置部署信息

编辑 _config.yml 文件，添加以下内容：

deploy:
  type: git
  repo: https://github.com/yourname/yourname.github.io.git
  branch: main

6.4 生成并部署

运行以下命令生成静态文件并部署到 GitHub：

hexo generate
hexo deploy

6.5 访问博客

打开浏览器，访问 https://yourname.github.io，查看你的博客。

7. 常用命令总结

以下是 Hexo 常用命令及其功能说明：

命令 功能
hexo new post 创建新文章
hexo generate 生成静态文件
hexo server 启动本地服务器
hexo deploy 部署到远程服务器
hexo clean 清理缓存和生成的静态文件
