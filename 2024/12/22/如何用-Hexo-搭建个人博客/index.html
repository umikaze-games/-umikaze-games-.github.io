如何用 Hexo 搭建个人博客

写博客是一种记录生活、分享知识的方式，同时也是展示个人能力的舞台。今天，我将以 Hexo 为例，带你完成从安装到部署的完整过程，帮助你搭建一个属于自己的博客。

1. 安装 Hexo

Hexo 是基于 Node.js 的静态博客框架，所以在开始之前，你需要完成以下准备：

1.1. 安装 Node.js

1.1.1. 打开 Node.js 官网。

1.1.2. 下载并安装最新版 Node.js（建议使用 LTS 版本）。

1.1.3. 安装完成后，在终端中运行以下命令检查版本：

node -v
npm -v

1.1.4. 如果输出类似以下内容，说明安装成功：

 v16.20.1
 8.19.2

1.2 安装 Git

1.2.1. 打开 Git 官网。

1.2.2. 下载并安装 Git。

1.2.3. 安装完成后，在终端中运行以下命令检查版本：

    git --version

1.1.4. 输出类似以下内容，说明安装成功：

git version 2.34.1

1.3 安装 Hexo

1.3.1. 在终端中运行以下命令全局安装 Hexo CLI：

npm install -g hexo-cli

1.3.2. 安装完成后，运行以下命令确认安装成功：

hexo -v

1.3.3. 输出类似以下内容，说明安装成功：

hexo: 6.3.0
hexo-cli: 4.2.0

2. 初始化博客

2.1 在终端中进入你希望存放博客的目录，例如：

cd ~/Documents

2.2 运行以下命令初始化一个新的 Hexo 项目：

hexo init my-blog
cd my-blog
npm install

2.3 结构说明：

scaffolds：用于创建文章的模板文件。
source：存放博客的主要内容（如文章和图片）。
themes：主题文件夹，存放博客的主题文件。
_config.yml：Hexo 的全局配置文件。
